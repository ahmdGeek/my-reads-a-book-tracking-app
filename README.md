# To start the Application

### `npm install`

# FEATURES:

- Categorize books into
  - Currently reading
  - Want to Read
  - Read
- Add new Books to the your library
- Remove books from your library

# Technologies Used:

- ReactJS
- React Router DOM
- React Hooks with functional component
